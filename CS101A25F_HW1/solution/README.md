# CS101A(H) Week 1 

## What you need to submit
Submit exactly **one file**: `student/student_solution.hpp`.
- Implement all required functions in this single header.
- No `main()` in your submission.
- Do **not** change function signatures.
- Do **not** add/remove any public functions.
- **Do not** use `using namespace std;` at global scope.

## How to compile and run small tests 
```bash
make public
./build/public_test
```
## How to run grader(for TA)
```bash
make public
./build/public_test
make grader
./build/grader
```
