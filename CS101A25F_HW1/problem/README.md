# CS101A(H) Week 1

## What you need to submit
Submit `.hpp` files for each problem.
- Implement all required functions in the header files.
- No `main()` in your submission.
- Do **not** change function signatures.
- Do **not** add/remove any public functions.
- **Do not** use `using namespace std;` at global scope.

## How to compile and run small tests 
```bash
make public
./build/public_test
```
